Sécurité et Réseaux

Computer Networking and Security

This course provides an introduction to fundamental concepts in the design and implementation of computer networks, their protocols, and applications. Topics to be covered include: overview of network architectures, applications, network programming interfaces (e.g., sockets), transport, congestion, the network data plane (packet forwarding, addressing) and control plane (routing algorithms,software defined networks, network management), wireless and mobile communication, cloud computing, and network security.
